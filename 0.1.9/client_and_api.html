Client#

The ApiClient object#

The Python client acts as an interface that communicates with the avatarization engine. For more information about the concepts and avatarization, checkout our main docs.

The ApiClient is the main interfaces that you should use. You’ll instantiate it, and authenticate using the credentials to the engine.

Note

The request() method is only there if you want to make a manual call to the engine. For regular use, use the methods described below.

exception avatars.client.FileTooLarge#
class avatars.client.ApiClient(base_url: str, timeout: Optional[int] = 5)#

Request the API.

Methods#

Here below are the methods provided that communicate with the engine. The API they expose uses pydantic objects to help you pass in the correct arguments to the methods.
